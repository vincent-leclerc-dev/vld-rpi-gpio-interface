# vld-rpi-gpio-interface
A python script to control GPIO of a Raspberry Pi 3b+

## prerequisite

### Equipment
A Raspberry Pi 3b+

### Intall GPIO library
```
$ sudo apt-get update
$ sudo apt-get install rpi.gpio
```

### Install Python
```
$ python --version
$ sudo apt-get install python
```

### Test Python
```
$ python
```
```
>>> print("hello\r\nworld!")
hello
world!
```
